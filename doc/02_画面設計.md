# SCR-001 ホーム画面

## 目的

会話開始前の設定を行う。

---

## 入力項目

### 利用者プロフィール

| 項目      | 入力方式      | 必須 |
| ------- | --------- | -- |
| 年齢      | 数値        | ○  |
| 仕事      | テキスト      | ○  |
| 住んでいる場所 | テキスト      | ○  |
| 趣味      | テキスト（複数可） | ○  |
| その他情報   | 複数行テキスト   | 任意 |

---

### AI女性プロフィール

| 項目      | 入力方式      | 必須 |
| ------- | --------- | -- |
| 年齢      | 数値        | ○  |
| 仕事      | テキスト      | ○  |
| 住んでいる場所 | テキスト      | ○  |
| 趣味      | テキスト（複数可） | ○  |
| その他情報   | 複数行テキスト   | 任意 |

---

### AI設定

| 項目     | 入力方式  |
| ------ | ----- |
| 性格     | プルダウン |
| 会話スタイル | プルダウン |
| 価値観    | プルダウン |
| 難易度    | プルダウン |

---

### 会話設定

| 項目    | 入力方式 |
| ----- | ---- |
| 最低会話数 | 数値入力 |
| 最大会話数 | 数値入力 |

説明：

* 最低会話数まではAIは会話を終了しない
* 最大会話数到達時は必ず終了する

---

## ボタン

### 女性プロフィールをランダム生成

プロフィール・性格・価値観・会話スタイルをランダム生成する。

---

### 会話開始

入力内容をもとに会話画面へ遷移する。

# SCR-002 会話画面

## 目的

AI女性との会話を行う。

---

## 画面構成

```text
────────────────────────────

女性プロフィール（折りたたみ表示）

────────────────────────────

会話履歴

────────────────────────────

音声入力ボタン

テキスト入力欄

送信ボタン

────────────────────────────

現在の会話数

例

4 / 15

────────────────────────────
```

---

## 表示項目

### 女性プロフィール

初期状態では折りたたむ。

表示内容

* 年齢
* 趣味
* 仕事
* 居住地
* その他情報

※性格・価値観・内部状態は表示しない。

---

### 会話履歴

表示形式

```text
AI
こんにちは。

あなた
こんにちは。
```

時系列順に表示する。

---

### 入力

#### 音声入力

マイクボタンを押して録音開始。

音声認識後、自動でテキスト入力欄へ反映する。

---

#### テキスト入力

利用者が自由入力できる。

---

#### 送信

AIへ送信する。

---

### 会話数

表示例

```text
7 / 20
```

---

### 会話終了

AIが終了判定した場合、自動で評価画面へ遷移する。

---

# UI設計方針

本システムは「会話に集中できるUI」を最優先とする。

情報量を必要最低限に抑え、利用者が実際の婚活に近い心理状態で会話できるデザインを目指す。

会話中は評価情報やAI内部状態を表示せず、会話終了後にのみ分析結果を提示する。

# 婚活AIトレーナー（Konkatsu AI Trainer）

> AIとのロールプレイを通じて、婚活におけるコミュニケーション能力を向上させるトレーニングアプリ

---

# プロジェクト概要

婚活では、初対面の相手との会話が成功を左右する重要な要素です。

本プロジェクトは、AIを婚活女性としてシミュレーションし、実際の婚活を想定した会話練習ができるWebアプリケーションを開発することを目的としています。

AIは単なるチャットボットではなく、一人の人物として性格・価値観・会話スタイル・感情を持ち、利用者との会話内容に応じて反応を変化させます。

会話終了後には、婚活アドバイザーとして会話内容を分析し、改善点や次回へのアドバイスを提供します。

---

# プロジェクトコンセプト

本アプリは「AIチャットアプリ」ではありません。

**AIとの自然な会話をシミュレーションできる「Conversation Engine（会話エンジン）」** を開発し、その最初のシナリオとして婚活を実装します。

将来的には以下のようなシナリオ追加も想定しています。

* 婚活
* 面接練習
* 営業ロールプレイ
* 接客練習
* 英会話
* 雑談トレーニング

---

# 開発目的

本アプリでは以下を目的とします。

* 初対面の会話に慣れる
* 自然な質問力を身につける
* 相手への共感力を高める
* 会話の改善点を客観的に把握する
* 婚活で実践できるコミュニケーション能力を養う

---

# MVP（最初に実装する機能）

* 利用者プロフィール入力
* AI女性プロフィール入力
* AI女性プロフィールランダム生成
* AIとの会話
* 音声入力（Speech to Text）
* AIによる自然な会話
* 会話終了判定
* AIによる会話評価

| 役割         | 内容                    |
| ---------- | --------------------- |
| フロントエンド    | React + TypeScript    |
| バックエンド     | Node.js + Express     |

| 音声認識       | OpenAI Speech to Text |
| データ保存（MVP） | ブラウザメモリ（State管理）      |

---

# AIアーキテクチャ

AIは役割ごとに分離して設計します。

```text
プロフィール生成AI
        │
        ▼
会話AI
        │
        ▼
評価AI
```

それぞれ独立した責務を持ちます。

| AI         | 役割              |
| ---------- | --------------- |
| プロフィール生成AI | 女性プロフィール生成      |
| 会話AI       | 婚活女性として自然に会話する  |
| 評価AI       | 婚活アドバイザーとして評価する |

# ドキュメント一覧

## ファイル名称ルール

設計書は `doc/` 配下に次の規則で配置します。

| ルール | 内容 |
| --- | --- |
| 形式 | `連番_日本語タイトル.md` |
| 連番 | `01` から昇順。欠番を作らない |
| タイトル | 日本語（カタカナ可）。英語クラス名は日本語へ置き換える |
| バージョン | 改訂版は末尾に `_V2` を付与（例: `16_AI状態管理設計_V2.md`） |
| 例外 | 製品名 `Cursor`、略語 `AI` / `API` はそのまま使用可 |
